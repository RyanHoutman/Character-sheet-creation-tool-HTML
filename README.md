A4.css is used for rendering inside a browser
The rest of the stylesheets are found in the "stylesheets" folder and are loaded in using styles.js script.
This ensures that changes to the stylesheets are made to all character sheets past, present and future.

index.html is a blank sheet with a bunch of elements. Copy paste them as you wish to create the sheet you want.
The basic structure is the header with the name and race, with three columns below. On the left is the first column
which is filled with prof bonus banners, your skills and stats, and some racial/background stuff.
The second column contains your spellslots, hp, attacks and whatever you want below.
The third column are your class abilities. But you are allowed and capable to change the order as you wish.

Code for characters contains all the code of previous sheets (some more up to date than others)
Empty sheets contains the pdf versions of them.


The rendering and printing only works if the html file is at the same place as the index.html file. So copy paste it there
The folder are just for storage.
For printing in edge: enable background graphics and first print to pdf and then print the pdf using your printer.
